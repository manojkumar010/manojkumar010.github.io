# Manoj Kumar - AI Agent Creator Portfolio

This is the personal portfolio website for Manoj Kumar, an AI/ML professional specializing in Generative AI, Speech & Audio Analytics, and NLP.

## Features

*   **About Me:** Detailed information about Manoj Kumar's background, education, and passion for AI.
*   **Skills:** A comprehensive list of technical skills, including programming languages, ML/DL frameworks, NLP tools, and more.
*   **Experience:** Highlights of professional experience, including key projects and achievements at HSBC.
*   **Projects:** Showcase of significant projects like iSTENO, Fraud Call Detection, Call Quality Monitoring System, and published PyPI libraries.
*   **Contact:** Easy access to contact information.

*   HTML5

## How to View Locally

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/manojkumar010/manojkumar010.github.io.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd manojkumar010.github.io
    ```
3.  **Open `index.html`:**
    Simply open the `index.html` file in your preferred web browser. You can usually do this by double-clicking the file.

## How to Deploy to GitHub Pages

This repository is already set up for GitHub Pages, as it follows the `username.github.io` naming convention. Any content pushed to the `main` branch will be automatically deployed.

1.  **Ensure your repository name is `manojkumar010.github.io`** (which it already is).
2.  **Push your changes to the `main` branch:**
    ```bash
    git add .
    git commit -m "Initial portfolio website setup"
    git push origin main
    ```
3.  **Access your website:**
    Your website should be live at `https://manojkumar010.github.io/` within a few minutes.
